# ee-db-cluster

DB agnostic db cluster implementation. The module manages connections to readonly, readwrite & writeonly db nodes. This module is used by the ee-orm module. 

## DB Connectors

- MySQL: ee-mysq-connection 

## installation

    npm install ee-db-cluster
